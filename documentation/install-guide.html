Spark Installation Guide

Spark consists of the following:

  • spark and uninstall executables, located in the top-level directory
  • .install4j
  • docs
  • jre (optional)
  • lib
  • logs
  • plugins
  • resources

Installation

    Windows 95/NT/2000/XP

    Run the Spark installer, located in the spark.exe file. The application will be installed to c:\Program Files\Spark by default.

    Macintosh OS X

    Download Spark, located in the spark.dmg file, and drag the application to your /Applications folder.
